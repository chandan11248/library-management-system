Welcome to the **Library Management System** project! 🌟  
This repository contains a simple **C++ console application** for managing library operations such as adding, searching, and issuing books.

---

## 📂 Repository Structure

The repository contains:

- **project1.cpp** : Main C++ source code file implementing the library system.  
- Other project-related files as needed.

---

## 🌟 Features

- Add new books to the library  
- Search for books by title or author  
- Issue and return books  
- Display all available books  
- Simple console-based interface  

---

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**

    - `git clone https://github.com/chandan11248/library-management-system.git`  
    - `cd library-management-system`

2. **Compile the C++ code:**

    - Using g++:  
      ```bash
      g++ project1.cpp -o library
      ```

3. **Run the executable:**

    - On Linux/Mac:  
      ```bash
      ./library
      ```
    - On Windows:  
      ```bash
      library.exe
      ```
